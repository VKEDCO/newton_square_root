- Newton's square root approximation method in Python & Perl.
Function abstraction is based on Chapter 01 of "Structure & 
Interpretation of Computer Programs"  by Abelson and Sussman.

- I used XEmacs on Windows to type newton_square_root.pl. Use 
dos2unix (http://sourceforge.net/projects/dos2unix/) to handle 
^M to run newton_square_root.pl on Unix/Linux.

- Screencasts available at http://www.youtube.com/v/OAmAwSV5EGk?version=3
and http://www.youtube.com/v/nDlip0Mg8tE?version=3.
